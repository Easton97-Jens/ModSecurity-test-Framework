# Generated Framework reports

**Language:** English | [Deutsch](README.de.md)

This directory is a navigation marker, not a second generated-report source.
The canonical Framework-generated Markdown is currently under
[`../testing/generated/`](../testing/generated/): coverage reports are under
`coverage/` and runtime reports are under `runtime/`.

Run `make refresh-framework-reports` (or the documented reporting target) to
refresh generated output. Do not manually edit a file marked generated; update
the reporting source under `ci/reporting/` and regenerate it instead.
