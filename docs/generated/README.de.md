# Generierte Framework-Berichte

**Sprache:** [English](README.md) | Deutsch

Dieses Verzeichnis ist ein Navigationsmarker, keine zweite Quelle generierter
Berichte. Kanonisches, vom Framework generiertes Markdown liegt derzeit unter
[`../testing/generated/`](../testing/generated/): Coverage-Berichte unter
`coverage/` und Runtime-Berichte unter `runtime/`.

`make refresh-framework-reports` (oder das dokumentierte Reporting-Target)
aktualisiert generierte Ausgabe. Eine als generiert markierte Datei nicht
manuell ändern; stattdessen die Reporting-Quelle unter `ci/reporting/`
aktualisieren und neu generieren.
