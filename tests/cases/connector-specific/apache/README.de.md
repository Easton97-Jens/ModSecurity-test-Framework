# Apache-Connector-spezifische Fälle

**Sprache:** [English](README.md) | Deutsch

Konnektorspezifische YAML-Fälle für Apache liegen hier, sofern verfügbar. Die
kanonischen späten Phase-4-Fälle prüfen im echten Host den Safe-Log-only- und
den Strict-Connection-Abort-Ausgang nach Response-Commit. Sie belegen keine
Pre-Commit-Response-Body-Deny-Semantik.
